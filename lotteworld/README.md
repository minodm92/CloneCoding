# 롯데월드 민속박물관 웹사이트 클론코딩

HTML, CSS, JavaScript를 활용해 **롯데월드 민속박물관 메인 페이지**를 클론코딩한 퍼블리싱 포트폴리오 프로젝트입니다.

> **본 프로젝트는 개인 학습 및 포트폴리오 용도로 제작되었습니다.**

---

## 📌 프로젝트 소개

롯데월드 민속박물관 웹사이트의 메인 페이지를 분석하여 HTML, CSS, JavaScript를 활용해 직접 구현한 클론코딩 프로젝트입니다.

레이아웃 구성부터 전시 및 교육 프로그램 안내, 공지사항, 이미지 hover 인터랙션까지 실제 웹사이트의 구조를 분석하고 구현하면서 웹 퍼블리싱 역량을 향상시키는 것을 목표로 제작하였습니다.

- **프로젝트명** : 롯데월드 민속박물관 웹사이트 클론코딩
- **작업 유형** : Web Publishing
- **제작 목적** : 퍼블리싱 실습 및 취업 포트폴리오

## ✨ 주요 기능

### 1. 전시 안내 인터랙션

- 상설전시 및 특별전시 콘텐츠 구성
- Flex를 활용한 전시 카드 가로 배치
- 이미지 영역에 overflow를 적용한 확대 효과
- hover 시 이미지가 자연스럽게 확대되는 애니메이션
- 전시 카드에 따라 배경 이미지와 제목이 변경되는 인터랙션

---

### 2. Header

- 로고 및 GNB 메뉴 구성
- 관련 사이트, 아이스링크, 로그인 유틸리티 메뉴 배치
- Position을 활용한 메뉴 영역 정렬
- 시맨틱 마크업 적용

---

### 3. Contents

- 운영 시간, 전시, 교육 프로그램, 소식 섹션 구현
- Flex 및 Grid를 활용한 섹션별 레이아웃 구성
- 전시 및 프로그램 이미지 카드 배치
- 공지사항 목록과 이용 문의 바로가기 영역 구현
- 배경 이미지와 텍스트를 활용한 배너 UI 구성

---

### 4. Footer

- 문의 및 롯데월드 소개 링크 구성
- 주소, 사업자 정보, 고객센터 정보 배치
- 계열사 및 관련 사이트 메뉴 구현
- Position과 가상 요소를 활용한 구분선 표현

## 📚 배운 점

이번 프로젝트를 진행하면서 실제 문화시설 홈페이지를 분석하고 이를 HTML, CSS, JavaScript로 직접 구현하는 경험을 할 수 있었습니다.

특히 Flex와 Grid를 함께 활용해 여러 형태의 콘텐츠를 배치하고, hover 및 mouseenter 이벤트로 전시 이미지와 배경이 변화하는 인터랙션을 구현하면서 사용자 경험을 고려한 화면 구성과 이벤트 처리 방식을 익힐 수 있었습니다. 또한 시맨틱 마크업과 섹션별 레이아웃 설계의 중요성을 다시 한번 경험할 수 있었습니다.
