# 경동 공식 웹사이트 클론코딩

HTML, CSS, JavaScript를 활용해 **경동 공식 웹사이트 메인 페이지**를 클론코딩한 퍼블리싱 포트폴리오 프로젝트입니다.

> **본 프로젝트는 개인 학습 및 포트폴리오 용도로 제작되었습니다.**

---

## 📌 프로젝트 소개

경동 공식 웹사이트의 메인 페이지를 분석하여 HTML, CSS, JavaScript를 활용해 직접 구현한 클론코딩 프로젝트입니다.

레이아웃 구성부터 UI 디자인, 슬라이드 인터랙션까지 실제 웹사이트의 구조를 분석하고 구현하면서 웹 퍼블리싱 역량을 향상시키는 것을 목표로 제작하였습니다.

- **프로젝트명** : 경동 공식 웹사이트 클론코딩
- **작업 유형** : Web Publishing
- **제작 목적** : 퍼블리싱 실습 및 취업 포트폴리오
